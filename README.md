### nonebot_plugin_emby_webhook
emby更新推送

打开emby设置里的通知选项，添加一个通知，名称随意，网址填http://127.0.0.1:15434/emby/webhook
其中 http://127.0.0.1:15434 为你的nonebot2运行的端口
通知类型选 媒体库 新媒体已添加
